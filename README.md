# manishverma-tech
Personal portfolio and professional site for Manish Verma (Lead Experience Engineer). Static GitHub Pages project with case studies, resume, and blog-ready templates.

---

### What’s included
- **Home / About / Work / Case Studies / Blog / Resume / Contact** — ready HTML templates.  
- **Project case studies** with role, problem, approach, impact, and assets.  
- **Static assets** in `/assets/` (images, OG images, resume PDF).  
- **SEO & social**: Open Graph metadata, `sitemap.xml`, `robots.txt`.  
- **GitHub Pages config**: base-path aware links and `404.html` SPA fallback.

---

### Quick start
1. **Clone** the repo.  
2. **Edit** content in the relevant folders (`/about/`, `/work/`, `/blog/`, `/resume/`).  
3. **Add** images to `/assets/` and update OG images.  
4. **Commit & push** to `main`.  
5. **Enable GitHub Pages** in repo settings (branch: `main`, folder: `/` or `/docs`).  
6. **Test** deep links and mobile layout; add `404.html` for SPA fallback if using client-side routing.

- `/resume/index.html` and `/resume/manish-verma-resume.pdf` — Resume + PDF  
- `/assets/` — images, css, js, pdfs  
- `/404.html`, `/sitemap.xml`, `/robots.txt`, `/CNAME` (if used)
